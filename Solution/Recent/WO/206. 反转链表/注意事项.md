## 心得

* 反转链表的标准流程

  ```python
  head # 待反转链表的头节点
  tmp # 临时变量
  prev # 空节点, 作为反转链表的末尾
  
  tmp = head.next # 必须保存头结点的下一个节点, 否则会有内存问题
  head.next = cur # 头结点的指向
  cur = head # 尾节点移位
  head = tmp # 头节点移位
  
  ```
